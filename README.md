# Banking Loan Eligibility System

## Description
The Banking Loan Eligibility System is a simple Java console application that determines whether a customer qualifies for a bank loan. The application collects the customer's name, age, annual salary, and credit score, then evaluates these details against predefined eligibility criteria. Based on the evaluation, it displays whether the loan is approved or rejected.

## Features
- Accepts customer details through the console
- Checks loan eligibility based on:
  - Age (21 years or above)
  - Salary (₹300,000 or above)
  - Credit Score (700 or above)
- Displays loan approval or rejection
- Beginner-friendly Java project demonstrating conditional statements and user input

## Technologies Used
- Java
- Scanner Class
- Conditional Statements (`if`)
